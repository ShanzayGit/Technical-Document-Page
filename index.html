Basics of HTML

Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It forms the backbone of every webpage and works alongside CSS and JavaScript to build complete web experiences.

HTML describes the structure of a web page using a series of ,elements, represented by tags. Browsers read these tags and render them as visible content.

Key Facts About HTML

  • First developed by Tim Berners-Lee in 1991
  • Current standard is HTML5 (maintained by WHATWG)
  • Files use the .html or .htm extension
  • HTML is not a programming language it is a markup language
  • Works on all browsers and operating systems
Basic HTML Document Structure

Every HTML document follows a standard structure. This structure tells the browser how to interpret and display the content.

Code Example

  
    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>My First Page</title>
      </head>
      <body>
        <h1>Hello, World!</h1>
        <p>Welcome to my first HTML page.</p>
      </body>
    </html>

Structure Breakdown

  • <!DOCTYPE html> Declares the document as HTML5
  • <html> The root element that wraps all conten
  • <head> Contains metadata (not visible on the page)
  • <title> Sets the browser tab title
  • <body> Contains all visible page content
HTML Elements and Tags

An HTML element consists of an opening tag, content, and a closing tag. Some elements are self-closing and don't need a closing tag.

Code Example


<!-- Standard element -->
<p>This is a paragraph.</p>

<!-- Self-closing elements -->
<img src="image.jpg" alt="A sample image" />
<br />
<input type="text" placeholder="Enter name" />

Tags Attributes

Tags can also carry attributes that provide additional information. For example:

  • src specifies the source of an image
  • href specifies the URL of a link
  • class and id are used for styling and scripting
  • alt provides alternative text for accessibility
Text and Heading Elements

HTML provides six levels of headings (<h1> through <h6>) and several tags for formatting text content.

Code Example


<h1>Main Page Title</h1>
<h2>Section Heading</h2>
<h3>Sub-section Heading</h3>

<p>This is a regular paragraph of text.</p>
<p>This text has a <strong>bold word</strong> and an <em>italic word</em>.</p>
<p>Visit <a href="https://example.com">this link</a> for more info.</p>

<blockquote>This is a blockquote for long quotes or references.</blockquote>

Common text elements and their purpose:

  • <h1> to <h6> Headings from most to least important
  • <p> Paragraph of body text
  • <strong> Bold text (semantic importance
  • <em> Italic text (emphasis)
  • <a> Hyperlink to another page or resource
  • <br> Line break within text
  • <blockquote> Indented quotation block
Lists in HTML

HTML supports three types of lists: unordered, ordered, and definition lists. Lists are used for navigation menus, feature lists, steps, and more.

Code Example


<!-- Unordered List -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Ordered List -->
<ol>
  <li>Open your code editor</li>
  <li>Create a new file</li>
  <li>Save it as index.html</li>
  <li>Write your HTML</li>
  <li>Open it in a browser</li>
</ol>

Types of HTML lists:

  • <ul> Unordered list (bullet points)
  • <ol> Ordered list (numbered)
  • <dl> Definition list (term + description pairs)
  • <li> Individual list item (used in <ul>; and <ol>)
  • <dt> / <dd> Definition term and description (used in <dl>)
HTML Forms and Input Elements

Forms allow users to input and submit data. They are essential for login pages, search bars, contact forms and more.

Code Example


<form action="/submit" method="POST">
  <label for="name">Full Name:</label>
  <input type="text" id="name" name="name" placeholder="John Doe" required />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" placeholder="you@example.com" />

  <label for="role">Role:</label>
  <select id="role" name="role">
    <option value="dev">Developer</option>
    <option value="design">Designer</option>
  </select>

  <button type="submit">Submit</button>
</form>

Commonly used form input types:

  • text Single-line text input
  • email Email address input with basic validation
  • password Masked text input for passwords
  • checkbox / radio Boolean or single-choice options
  • submit Button that sends the form data
